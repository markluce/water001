## 二、檔名解讀

檔名每個部分都有特定含義：

| 部分 | 含義 |
|------|------|
| **triton** | Triton Poker（高額錦標賽系列賽） |
| **one** | 某一特定賽事 |
| **mystery** | Mystery Event（神秘賽事） |
| **5left** | 剩餘 5 位玩家（接近決賽桌） |
| **CO** | 主角位置：Cutoff（關煞位） |
| **44** | 主角手牌：口袋 4（4♠4♥） |
| **5bb** | 約 5 大盲注的有效碼量（檔名標示，實際主角有 60.8bb） |
| **nodelock** | 已套用 Nodelock（針對特定對手調整的非均衡策略） |
| **.hrcz** | HRC 求解器壓縮檔格式 |

## 三、檔案內部結構

### 1. settings.json — 賽事與求解器設定

```json
{
  "handdata": {
    "stacks": [2000, 580, 7300, 2300, 4600],
    "blinds": [120, 60, 120],

```

**解讀：**
- 5 位玩家的籌碼量：2000、580、7300、2300、4600

- 大盲前注制（Big Blind Ante）

### 2. 賽事結構（KO 淘汰賞金賽）

```json

  "bounty": 8000.0,
  "prizes": {
    "1": 395000,
    "2": 240000,
    "3": 169000,
    "4": 127000,
    "5": 99000
  }
}
```

**解讀：**
- KO（Knockout）淘汰賞金賽制
- 每次淘汰對手可獲得 $8,000 賞金
- 獎金池總計 $1,030,000
- 第一名 $395,000（38.3%）
- 第五名 $99,000（9.6%）

### 3. state.json — ICM 權益計算

| BTN (2300 chips) | $198,452 | $200,145 | +0.85% |
| BB (4600 chips) | $251,926 | $253,200 | +0.51% |

HJ 位僅 580 chips（4.8bb），ICM 權益大幅下降 3.95%，面臨極大淘汰壓力。

## 四、策略意義總結

1. **場景**：Triton Poker Mystery Event，5 人桌，接近泡沫或決賽桌
2. **主角**：CO 位持有 4♠4♥，60.8bb 深碼，是桌上最大碼量
3. **關鍵對手**：HJ 僅 4.8bb，面臨極大 ICM 壓力
4. **賽制影響**：$8,000 淘汰賞金（佔第五名獎金的 8.1%）使得對短碼玩家更積極的策略成為正期望值
5. **Nodelock**：策略已針對特定對手傾向調整，非純 GTO 均衡解，而是剝削策略
6. **ICM 結論**：主角權益極穩定（僅 +0.04%），不需冒險；HJ 是主要被壓迫對象
